AES

AES (Advanced Encryption Standard) is a symmetric key cipher which is currently the industry standard. It is a block cipher and uses multiple rounds of a substitution permutation network algorithm to encrypt data.

Triple DES

Triple DES is a symmetric key cipher which uses the original Data Encryption Standard (DES) three times to encrypt data. It was used widely in the late nineties until it was superseded by AES. The small key of size of DES made it vulnerable to brute-force attacks and Triple DES was a simple method of increasing the security of the cipher, however, it is still not as secure as AES.

RC4

RC4 is a symmetric stream cipher and is one of the most commonly used stream ciphers. It is used in the Secure Socket Layer(SSL)/Transport Layer Security (TLS) protocols and Wi-Fi Security Protocl WEP due to its ease of use and performance speed. However, it does have severe vulnerabilities and cannot be used on tiny bits of data so it is not used widely anymore.

Chaos Map

Chaotic systems are a sub-type of nonlinear dynamical systems and can produce unpredictable/chaotic behaviour over time. The logistic map is a type of chaos map which maps the population value at any time step to its value at the next time step.

Go

Steganography

Image steganography is the art of hiding secret messages in images. The bits of the secret message are embedded into some of the pixels which make up the image. Since only one of the three RGB values are changed, the image looks the same before and after encoding.
